En este repositorio se recogen las prácticas correspondientes a la parte de **Robótica Móvil** de la asignatura **Ampliación de Robótica**, realizadas por la alumna **Paula Mancilla Pulido** durante el curso **2025/2026** de **GIERM**.

## Contenido

El repositorio incluye los siguientes paquetes:

- `seg_tray`
- `seg_tray_msgs`

Ambos paquetes pueden utilizarse en cualquier workspace de ROS 2 copiándolos dentro de la carpeta `src`.
